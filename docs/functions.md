### Size Functions

Return a quotient/product of the provided value.

```sass
div
  // padding: 3px
  padding: quarter(12px)

div
  // padding: 4px
  padding: third(12px)

div
  // padding: 6px
  padding: halve(12px)

div
  // padding: 24px
  padding: double(12px)

div
  // padding: 36px
  padding: triple(12px)

div
  // padding: 48px
  padding: quadruple(12px)
```
