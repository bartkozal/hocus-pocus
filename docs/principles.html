  1. Follow the rules from Mark Otto's Code Guide
  2. Use autoprefixer or autoprefixer-rails

  3. Use the following naming convention:

    {% highlight html %}
    {% endhighlight %}

  4. Avoid styling generic HTML elements:

    {% highlight css %} /* bad */ p { ... } /* good */ .blog-post-body > p { ... } {% endhighlight %}

  5. Avoid nested classes:

    {% highlight css %} /* bad */ .blog-post .meta .author { ... } /* better */ .blog-post-meta-author { ... } /* good */ .post-author { ... } {% endhighlight %}

  6. Never use the HTML id attribute in stylesheets:

    {% highlight css %} /* bad */ #headline { ... } /* good */ .headline { ... } {% endhighlight %}
