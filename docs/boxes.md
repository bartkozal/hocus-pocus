Use `.box` to create an element out of your content. The last element inside
has no bottom margin.

<div class="example">
  <div class="box" style="background-color: #eee;">
    <p>Element inside a box</p>
  </div>
</div>

```html
<div class="box" style="background-color: #eee;">
  ...
</div>
```
