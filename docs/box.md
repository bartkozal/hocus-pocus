Use `.box` to create an element out of your content. It has default
padding background equal the value of `$color-ui` and removes bottom
margin from a last element inside it.

<div class="example">
  <div class="box">
    <p>Element inside a box.</p>
  </div>
</div>

```html
<div class="box">
  ...
</div>
```
