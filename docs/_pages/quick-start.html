Quick Start

Hocus-Pocus requires Sass (version 3.3.0 or later) to proper work.

Bower

Bower is a recommended way to install:

{% highlight sh %} $ bower install hocus-pocus {% endhighlight %}

Then import a main file just after your defaults, but before rest of stylesheets. If you want to use color variables in your settings import them first.

{% highlight scss %} // Define your settings $font-family-base: "Open Sans", sans-serif; $grid-container: 70rem; // Import framework @import "bower_components/hocus-pocus/hocus-pocus"; // Import your stylesheets @import "compontents/meter", "compontents/price-box", "inbox/landing-headings-fix"; {% endhighlight %}

npm

If you prefer Hocus-Pocus is available as a npm package:

{% highlight sh %} $ npm install hocus-pocus {% endhighlight %}
