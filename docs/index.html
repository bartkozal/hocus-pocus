Hocus-Pocus is a simple and design-free SASS micro framework, ideal as a CSS starter kit.

It could not exist without work of awesome people behind these projects:

  • inuitcss
  • normalize.css
  • flexboxgrid
  • sassy-gridlover
  • colors

Quick Start

To install Hocus-Pocus use Bower:

{% highlight sh %} $ bower install hocus-pocus {% endhighlight %}

Or Rails Assets if you have a Ruby app:

{% highlight ruby %} # Gemfile source 'https://rails-assets.org' do gem 'rails-assets-hocus-pocus' end {% endhighlight %}

Then include the main Hocus-Pocus file after your variables, but before the rest of stylesheets:

{% highlight sass %} $base-font-family: "Open Sans", sans-serif $grid-container: 70rem @import "bower_components/hocus-pocus/hocus-pocus" @import "objects/meter" @import "objects/price-box" @import "inbox/landing-headings-fix" {% endhighlight %}
